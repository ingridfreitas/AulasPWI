FUNÇÃO

!DOCTYPE

Mostra qual versão do html que está sendo utilizada.

p

Cria um parágrafo, que é onde se localiza nosso texto.

header

Cria um cabeçalho em qualquer seção.

h

Cria títulos para a página, podendo ser númerada de h1 a h6, tendo como diferença apenas o tamanho.

html

Elemento básico da estrutura. Todo o documento estará contido na tag do html.

span

Possui função semelhante ao do parágrafo (p), porém, ela é mais utilizada para colocar pequenos textos, como legendas por exemplo.

main

Representa o assunto principal do texto, onde o assunto se relaciona com o tema.

img

Não necessita de fechamento, e permite que coloquemos alguma imagem na nossa página.

head

Delimita todo o cabeçalho do documento, e não é visível para o leitor.

pre

Usada para colocar textos recém-formatados, principalmente códigos.

footer

Cria uma "anotação" no final do texto, chamado de rodapé.

ol

Cria uma lista organizada por números ordinais (1°, 2°...)

title

Cria um nome para a nossa janela, guia ou aba

strong

Deixa o texto em negrito e não quebra linha, permitindo assim a formatação do texto.

article

Cria um artigo para a página, pricipalmente usado para blogs.

ul

Cria uma lista organizada por tópicos.

meta

Permite que a nossa aba, janela ou guia possua acento, permitindo o fácil entendimento do leitor.

i

Deixa o texto em itálico, e assim como o negrito, não quebra linha, permitindo a formatação.

nav

Define um conteúdo de navegação, muito utilizado na criação de listas e menus.

li

Onde fica os elementos da lista.

body

É o corpo do nosso texto e toda a parte visível do documento.

br

Essa tag não necessita de um fechamento e a sua função é quebrar linha,

div

Cria uma separação para a página, ajudanso a organizá-la melhor.

a

Cria um link na nossa página, permitindo que façamos ligação do nosso documento a outro.

Atributos

id

Atribui um id ao elemento, sendo ele único e específico dele;

height

Define uma altura para o elemento.

style

É usado para inserir CSS (estilo) de ntro da tag, como por exemplo, mudar de cor.

width

Define uma largura para o elemento.

class

Atribui uma classe, podendo ser usada por um ou mais elementos.

lang

Define qual linguagem será usada no documento.
